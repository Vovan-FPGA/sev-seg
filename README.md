# sev-seg

## Семисегментный индикатор


## Модули проекта

### 1. b_filter - Фильтр импульсов

Модуль предназначен для детектирования последовательности импульсов и формирования выходных сигналов.

**Параметры:**
- `CN` (default: 4) - количество бит для счетчика

**Входы:**
- `clk` - тактовый сигнал
- `in_b` - входной импульс
- `rst` - сброс (активный низкий уровень)

**Выходы:**
- `out_b` - выходной сигнал, активный при заполнении регистра сдвига
- `im_out_b` - импульсный выход, активный при детектировании фронта 

**Принцип работы:**
- Использует сдвиговый регистр `syn` для запоминания последних `CN` значений входного сигнала
- `out_b` становится активным, когда все биты регистра `syn` равны 1
- `im_out_b` генерирует короткий импульс при обнаружении паттерна "01" на позициях (CN-1, CN-2)

### 2. sev_seg_m - Контроллер семисегментного индикатора

Модуль для управления 4-разрядным семисегментным индикатором с общим катодом.

**Параметры:**
- `CN` (default: 20) - разрядность счетчика мультиплексирования

**Входы:**
- `clk` - тактовый сигнал
- `rst` - сброс (активный низкий уровень)
- `reg_seg[15:0]` - 16-битное значение для отображения (4 цифры по 4 бита)

**Выходы:**
- `SEG[7:0]` - сегменты индикатора (активный низкий уровень)
- `AN[3:0]` - аноды (общие катоды) для выбора разряда (активный низкий уровень)

**Принцип работы:**
- Демультиплексирует 4 цифры из входного 16-битного слова
- Использует динамическую индикацию с частотой переключения разрядов
- Преобразует BCD-код в 7-сегментный код (активный низкий уровень)
- Кодировка сегментов (активный низкий уровень, 1 - выключен, 0 - включен):
  - `SEG[0]` - сегмент A
  - `SEG[1]` - сегмент B
  - `SEG[2]` - сегмент C
  - `SEG[3]` - сегмент D
  - `SEG[4]` - сегмент E
  - `SEG[5]` - сегмент F
  - `SEG[6]` - сегмент G
  - `SEG[7]` - десятичная точка

### 3. sev_seg - Верхнеуровневый модуль

Объединяет фильтр и контроллер индикатора для создания законченного устройства.

**Входы:**
- `clk` - тактовый сигнал
- `rst` - сброс (активный низкий уровень)
- `KEY_SW` - кнопка/переключатель для подсчета импульсов

**Выходы:**
- `SEG[7:0]` - сегменты индикатора
- `AN[3:0]` - аноды индикатора

**Принцип работы:**
- Фильтрует входные импульсы от дребезга контактов
- Увеличивает счетчик `reg_seg` при каждом обнаруженном импульсе
- Отображает текущее значение счетчика на 4-разрядном индикаторе

## Тест

### tb_b_filter
Тестирует работу фильтра импульсов:
- Подает последовательность импульсов
- Проверяет формирование выходных сигналов `out_b` и `im_out_b`
- Генерирует VCD-файл для анализа временных диаграмм

### tb_sev_seg_m
Тестирует работу контроллера индикатора:
- Подает тестовые данные 0x3210 на вход
- Проверяет правильность мультиплексирования и преобразования в сегментный код
